How to Use HTML with CSS

  • Inline CSS – Add styles directly inside an HTML tag using the style attribute.
  • Internal CSS – Write CSS inside a <style> tag in the HTML <head> section.
  • External CSS – Link an external .css file using the <link> tag in the <head>.
  • Class Selectors – Use class attributes in HTML and style them in CSS using a dot (e.g., .box).
  • ID Selectors – Use id attributes in HTML and style them in CSS using a hash (e.g., #main).
